# Projecttitle-contextual-display
Choose different view for home and subfolders

They explained to me that there’s a way to set a different view for the home screen and for subfolders by using profiles. But I couldn’t get it to work, and honestly I don’t like that approach — it feels cumbersome. I preferred to create a patch that enables this feature directly in the projecttitle menu (since that’s what I’m using). So this is a patch specifically for projecttitle.

The patch adds a new option to the menu: “Use different mode for home and subfolders”, which can be toggled on or off, along with two additional menu entries to set the Home Folder Mode and the Subfolder Mode.
